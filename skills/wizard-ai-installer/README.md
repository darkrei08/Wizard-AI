# 📦 The Installer (/install)

**What is it?**
Meta-skill to autonomously install, integrate, and configure new skills, tools, and repositories into the Wizard-AI ecosystem. Use this when the user asks you to 'add a skill', 'install a tool', or provides a GitHub link/command for a new project.

**How to remember it:**
Drops a new skill, tool, or repo into the ecosystem with zero friction. The plug-and-play socket.

**Slash Command:** `/install`
