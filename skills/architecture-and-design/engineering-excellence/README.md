# 🏆 The Quality Gate (/excellence)

**What is it?**
Framework-agnostic software engineering standard for AI agents — SDD/TDD workflow, architecture principles, security, performance, accessibility, i18n, SEO, and production readiness. Load when implementing, reviewing, or hardening software in any language or stack.

**How to remember it:**
TDD + SOLID + Clean Code + Quality Gates. The bar you must clear before claiming 'done'.
