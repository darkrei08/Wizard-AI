# 🔎 Serena

**What is it?**
Use Serena for semantic code search, symbol navigation, and LSP-powered code understanding within a project. Use when you need to find all usages of a function, understand how a module is wired together, trace call graphs, or do semantic refactoring across a large codebase. Available via uvx (no install needed) as both a CLI and MCP server.

**How to remember it:**
Semantic code search via LSP. Find usages, trace call graphs, navigate symbols.
