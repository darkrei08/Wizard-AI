# Wizard-AI Core

This repository is the central nervous system of the Wizard-AI environment.
By installing this as a skill, Pi inherits the 5-step Loop Engineering Pipeline and access to all CLI tools (`wz-ai graph`, `wz-ai compress`, `wz-ai squeeze`, etc.).

## Available Commands
- `wz-ai help`: Central hub for all AI commands
- `wz-ai usage`: View token budget
- `wz-ai graph`: Map codebases to knowledge graphs
- `wz-ai compress`: Compress context via LLMLingua
- `wz-ai squeeze`: Compress CLI outputs
- `wz-ai convert`: Extract Markdown from PDF/DOCX
- `wz-ai mem`: Persistent semantic memory
- `wz-ai scaffold`: Scaffold projects
- `pi-cockpit-setup`: Configure Cockpit Tools Proxy and LLM models

## Loop Engine
When responding to complex prompts, the system automatically routes through:
1. `loop-1-plan`
2. `loop-2-develop`
3. `loop-3-debug`
4. `loop-4-refactor`
5. `loop-5-release`
