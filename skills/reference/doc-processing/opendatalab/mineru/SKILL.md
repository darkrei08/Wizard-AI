# MinerU Document Parsing

MinerU is a high-accuracy document parsing engine that converts PDF, DOCX, PPTX, XLSX, and Images into structured Markdown / JSON. It uses a VLM+OCR dual engine.

## When to use this skill
- When the user asks to parse a complex PDF, PPTX, or DOCX.
- When you need to extract tables, mathematical formulas, or layouts from documents.
- When standard markdown conversion tools fail.

## How to use MinerU
You can invoke MinerU via its CLI wrapper.

```bash
ai-mineru <path-to-document>
```
Or you can use the raw binary `mineru` or `magic-pdf` if the wrapper delegates to it.

## Verification
Always verify the output markdown file generated by MinerU to ensure the extraction quality meets the user's expectations.
