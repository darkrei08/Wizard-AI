# 🥷 Shadow Clone Jutsu (/shadow-clone)

**What is it?**
Use when facing 2+ independent tasks that can be worked on without shared state or sequential dependencies

**How to remember it:**
Like Naruto: clone yourself into N parallel agents, each solving a piece, then absorb their memories when they despawn.

**Slash Command:** `/shadow-clone`

**Part of:** Tier 1 — Supreme Orchestrator
