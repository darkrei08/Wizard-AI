# /gemini-usage

GeminiUsage tracks token and context usage for Gemini CLI sessions.
Installed at `~/.local/bin/gemini-usage`. Source in `~/.ai-skills/geminiusage/`.

## CLI Usage

```bash
# Show current session token usage
gemini-usage

# Show usage for a specific session/log file
gemini-usage --session ~/.gemini/logs/session-latest.json

# Show usage stats (tokens in/out, cost estimate)
gemini-usage --stats

# Monitor usage in real-time
gemini-usage --watch

# Export usage report
gemini-usage --export usage-report.csv

# Show help
gemini-usage --help
```

## What It Shows

- **Input tokens**: tokens consumed in your prompts
- **Output tokens**: tokens generated by the model
- **Context window**: current fill level (e.g., 45,230 / 1,000,000)
- **Estimated cost**: based on model pricing
- **Session history**: usage across multiple conversations

## When to Use

- User asks "how many tokens have I used?"
- User wants to monitor context window consumption
- User wants to audit AI costs across sessions
- User is approaching context limits and wants to know how close
- Building token-budget-aware workflows

## Notes

- Requires Gemini CLI to be installed and sessions to be logged
- Works with `agy` (Antigravity) sessions automatically
