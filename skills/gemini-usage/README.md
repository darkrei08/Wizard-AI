# 📊 Gemini Usage Tracker

**What is it?**
Use to track Gemini CLI token usage, context window consumption, and session costs from the terminal. Use when the user wants to monitor how many tokens have been used, check remaining context budget, or audit usage across sessions.

**How to remember it:**
