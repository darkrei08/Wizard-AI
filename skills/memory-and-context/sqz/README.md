# 📦 SQZ (DevOps copy)

**What is it?**
Use to compress verbose CLI output, JSON payloads, logs, and agentic session data to reduce token usage. Use when piping large command output into an LLM, when a tool produces verbose JSON, or when you want to compress structured data before including it in context. Also provides an MCP server for integration.

**How to remember it:**
Compress verbose CLI output and JSON payloads.
