# ⚡ Rust Token Killer (/rtk)

**What is it?**
RTK (Rust Token Killer) — CLI output compression proxy for AI agents. Intercepts shell command outputs (git, npm, ls, grep, etc.) and compresses them 60-90% before they enter the LLM's context window. Single Rust binary, zero deps, <10ms latency. Use when CLI output is verbose and wastes tokens.

**How to remember it:**
A Rust binary that intercepts shell output and compresses it 60-90% in <10ms. Zero deps, pure speed.
