# ⚡ FlashRank

**What is it?**
Use to re-rank documents or passages by relevance before inserting them into LLM context — reduces wasted tokens by putting the most relevant chunks first. Use in RAG pipelines, when the user has a list of documents to filter, or when context needs to be pruned to top-K most relevant passages.

**How to remember it:**
Re-rank documents by relevance before LLM ingestion. Top-K most relevant first.
