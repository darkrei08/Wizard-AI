# 📝 MarkItDown

**What is it?**
Use to convert any file (PDF, DOCX, XLSX, PPTX, HTML, images, audio, ZIP, EPUB, XML, CSV, JSON) into clean Markdown suitable for LLM ingestion or RAG. Use when the user wants to analyze a non-text file, when a document needs to be added to context, or before running Graphify on a mixed-format corpus.

**How to remember it:**
Convert any file (PDF, DOCX, PPTX, HTML, images) to clean Markdown.
