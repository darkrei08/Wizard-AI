# /kinetics-ui

Kinetics is a gallery of 117 spring-physics micro-interactions designed to elevate the web UI.

## When to use:
- When the user asks for premium, modern, fluid animations.
- During Loop 2 frontend component implementation (React, Vue, Astro, HTML/CSS).
- When applying `taste-skill` and `frontend-design`.

## Core Concepts:
- **Spring Physics:** Instead of standard linear or ease-in-out transitions, use spring physics for natural, bouncy, and responsive feeling interactions.
- **Micro-interactions:** Focus on hover states, click feedback, toggle switches, and loading states.
- **Framer Motion / CSS:** In React, prefer `framer-motion` with spring configurations (e.g., `type: "spring", stiffness: 300, damping: 20`). In CSS, use `linear()` easing functions or specialized libraries if available.

## Integration:
Inject kinetics principles when generating UI artifacts or when writing code in `02. loop-2-develop`. Combine with `mengto-skills` for a complete UX overhaul.
