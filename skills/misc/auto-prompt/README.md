# ✍️ The Prompt Polisher (/prompt)

**What is it?**
Prompt Middleware skill. Automatically rewrites and structures messy natural language inputs into optimized XML prompts (using Simon Willison's llm CLI). Use when delegating tasks, refining user instructions, or preparing complex prompts to reduce LLM hallucinations and enforce skill constraints.

**How to remember it:**
Your messy thought goes in, a crystal-clear XML prompt comes out. The middleware between your brain and the LLM.

**Slash Command:** `/prompt`

**Part of:** Tier 1 — Supreme Orchestrator
