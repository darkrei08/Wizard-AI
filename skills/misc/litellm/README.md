# 🔀 LiteLLM Router

**What is it?**
Use LiteLLM to proxy, route, or track costs across 100+ LLM providers (OpenAI, Anthropic, Gemini, Mistral, etc.) with a unified API. Use when the user wants to switch providers, track token costs, run a local proxy, or call multiple LLMs with the same interface.

**How to remember it:**
Unified API for 100+ LLM providers. Switch, track costs, proxy.
