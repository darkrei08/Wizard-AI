# Wizard-AI Core

This repository is the central nervous system of the Wizard-AI environment.
By installing this as a skill, Pi inherits the 5-step Loop Engineering Pipeline and access to all CLI tools (`ai-graph`, `ai-compress`, `ai-squeeze`, etc.).

## Available Commands
- `ai-help`: Central hub for all AI commands
- `ai-usage`: View token budget
- `ai-graph`: Map codebases to knowledge graphs
- `ai-compress`: Compress context via LLMLingua
- `ai-squeeze`: Compress CLI outputs
- `ai-convert`: Extract Markdown from PDF/DOCX
- `ai-mem`: Persistent semantic memory
- `ai-scaffold`: Scaffold projects
- `pi-cockpit-setup`: Configure Cockpit Tools Proxy and LLM models

## Loop Engine
When responding to complex prompts, the system automatically routes through:
1. `loop-1-plan`
2. `loop-2-develop`
3. `loop-3-debug`
4. `loop-4-refactor`
5. `loop-5-release`
