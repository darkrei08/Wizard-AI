# /gemini-usage

GeminiUsage tracks token and context usage for Gemini CLI (Antigravity) sessions.

Available as both `gemini-usage` (base command) and `wz-ai-usage` (wrapper alias) — both are installed and interchangeable.
Source in `~/.wizard-ai/geminiusage/`.

## CLI Usage

```bash
# Show current session token usage
wz-ai-usage

# Show usage for today
wz-ai-usage --today

# Show last 24 hours of usage
wz-ai-usage --rolling-24h

# Check quota status
wz-ai-usage --quota

# Full usage history
wz-ai-usage --history

# Show help
wz-ai-usage --help

# Also works with base command
gemini-usage --stats
```

## What It Shows

- **Input tokens**: tokens consumed in your prompts
- **Output tokens**: tokens generated by the model
- **Context window**: current fill level (e.g., 45,230 / 1,000,000)
- **Estimated cost**: based on model pricing
- **Session history**: usage across multiple conversations

## When to Use

- User asks "how many tokens have I used?"
- User wants to monitor context window consumption
- User wants to audit AI costs across sessions
- User is approaching context limits and wants to know how close
- Building token-budget-aware workflows

## Notes

- Requires Gemini CLI (Antigravity) to be installed and sessions to be logged
- Works with `agy` sessions automatically
